Escola

Uma escola forneceu uma base de dados com as informações das notas de seus alunos, e agora deseja obter a média final de cada aluno e gerar um relatório com as informações. Sabendo que a média mínima para que o aluno seja aprovado é 6.0 outros dois relatório devem ser gerados listando os alunos reprovados e aprovados. Utilize os métodos map e filter para gerar os relatórios necessários.
